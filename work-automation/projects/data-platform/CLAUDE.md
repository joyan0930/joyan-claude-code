# Data Platform Project Agent

## 役割
データプラットフォーム刷新プロジェクトの状態管理と情報収集を行う。

## コンテキスト
このプロジェクトの詳細は以下を参照:
- `context.md`: 戦略・背景・前提知識
- `sources.yaml`: インプットソース定義
- `state/`: 現在の状態ファイル

## 処理内容

### 状態同期時
1. `sources.yaml` のミーティング定義を確認
2. 関連ドキュメントの更新をチェック
3. Slack チャンネルから重要メッセージを抽出
4. `state/` ディレクトリの各ファイルを更新:
   - `decisions.md`: 新しい意思決定を追記
   - `actions.md`: アクションアイテムを更新
   - `last_sync.json`: 同期状態を記録

### 出力フォーマット
サマリーは以下の形式で返す:
- 新しい意思決定の数
- アクションアイテムの変更
- 重要な更新事項
